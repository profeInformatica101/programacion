Cheat Sheet Java: Ficheros de Texto

Guía rápida para leer, escribir y gestionar archivos de texto en Java.

java.nio.file.Files Path BufferedReader BufferedWriter UTF-8

Imports más usados 

import java.io.*;
import java.nio.file.Files;
import java.nio.file.Path;
import java.nio.charset.StandardCharsets;
import java.nio.file.StandardOpenOption;
import java.util.List;

Idea clave

  • Path: representa la ruta del archivo.
  • Files: permite leer, escribir y comprobar archivos.
  • BufferedReader: lectura línea a línea.
  • BufferedWriter: escritura eficiente de texto.
  • UTF-8: codificación recomendada.

Métodos más importantes

Elemento Uso Ejemplo
Path.of("archivo.txt") Crea una ruta Path ruta = Path.of("datos.txt");
Files.readString(path) Lee todo el texto String texto = Files.readString(ruta);
Files.readAllLines(path) Lee todas las líneas List<String> lineas = Files.readAllLines(ruta);
Files.writeString(path, texto) Escribe texto Files.writeString(ruta, "Hola");
Files.exists(path) Comprueba si existe if (Files.exists(ruta)) { ... }
BufferedReader Lee línea a línea br.readLine()
BufferedWriter Escribe línea a línea bw.write("texto")

Leer un archivo completo 

import java.nio.file.Files;
import java.nio.file.Path;

public class LeerArchivo {
    public static void main(String[] args) {
        try {
            Path ruta = Path.of("datos.txt");
            String contenido = Files.readString(ruta);
            System.out.println(contenido);
        } catch (Exception e) {
            System.out.println("Error al leer el archivo: " + e.getMessage());
        }
    }
}

Escribir un archivo 

import java.nio.file.Files;
import java.nio.file.Path;

public class EscribirArchivo {
    public static void main(String[] args) {
        try {
            Path ruta = Path.of("salida.txt");
            Files.writeString(ruta, "Hola desde Java");
            System.out.println("Archivo guardado correctamente.");
        } catch (Exception e) {
            System.out.println("Error al escribir: " + e.getMessage());
        }
    }
}

Leer línea a línea 

import java.io.BufferedReader;
import java.io.FileReader;

public class LeerLineas {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("datos.txt"))) {
            String linea;
            while ((linea = br.readLine()) != null) {
                System.out.println(linea);
            }
        } catch (Exception e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

Añadir texto al final 

import java.nio.file.Files;
import java.nio.file.Path;
import java.nio.file.StandardOpenOption;

public class AnadirTexto {
    public static void main(String[] args) {
        try {
            Path ruta = Path.of("salida.txt");
            Files.writeString(
                ruta,
                "\nNueva línea añadida",
                StandardOpenOption.CREATE,
                StandardOpenOption.APPEND
            );
            System.out.println("Texto añadido.");
        } catch (Exception e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

Ejemplo completo 

import java.nio.file.Files;
import java.nio.file.Path;
import java.util.List;

public class EjemploFicheros {
    public static void main(String[] args) {
        try {
            Path ruta = Path.of("alumnos.txt");

            // Escribir contenido inicial
            Files.writeString(ruta, "Ana\nLuis\nMarta");

            // Leer todas las líneas
            List<String> alumnos = Files.readAllLines(ruta);

            System.out.println("Listado de alumnos:");
            for (String alumno : alumnos) {
                System.out.println("- " + alumno);
            }

        } catch (Exception e) {
            System.out.println("Se produjo un error: " + e.getMessage());
        }
    }
}

Errores frecuentes

  • No usar try-catch.
  • Escribir una ruta incorrecta.
  • No saber si el archivo se sobrescribe o se añade.
  • Olvidar cerrar flujos en clases clásicas de java.io.
  • No tener en cuenta la codificación del texto.
Consejo: para ejemplos sencillos y modernos en clase, usa Path + Files. Para enseñar el funcionamiento interno, muestra también BufferedReader y BufferedWriter.
