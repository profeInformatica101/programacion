Persistencia

JPA (Jakarta Persistence API) es el estándar ORM para mapear objetos Java con bases de datos relacionales, evitando SQL manual en la lógica de negocio.

Especificación oficial Jakarta Persistence

Entidades

Una entidad representa una tabla en la base de datos.

import jakarta.persistence.*;

@Entity
@Table(name="empleados")
public class Empleado {

  @Id
  @GeneratedValue(strategy = GenerationType.IDENTITY)
  private Long id;

  private String nombre;
}
@Entity API

Operaciones CRUD

  • Create: persist()
  • Read: find()
  • Update: merge()
  • Delete: remove()
em.persist(obj);
em.find(Empleado.class, 1);
em.merge(obj);
em.remove(obj);
