Tema 6. Cadenas de caracteres

  1. Programación en Java
  2. Tema 6 · Cadenas de caracteres

Una cadena es una secuencia de caracteres. Las cadenas sirven principalmente para mostrar, almacenar y leer texto. Pueden contener cualquier tipo de caracteres, incluyendo dígitos numéricos y signos de puntuación.

En Java los valores de tipo cadena se delimitan por comillas dobles, por ejemplo: "Hola mundo".

En Java, las cadenas se representan mediante objetos de la clase String. Esta clase tiene una característica especial: para crear un objeto no es necesario usar new(); basta con escribir la cadena entre comillas y el compilador llama automáticamente al constructor de String.

El operador + concatena cadenas, igual que el método concat().

Al ser una clase, las variables de tipo String son por referencia. Si no se inicializan contienen null.

Para comparar cadenas se debe usar equals(). El operador == comprueba si dos variables apuntan al mismo objeto en memoria. El método equalsIgnoreCase() permite comparar ignorando mayúsculas y minúsculas.

Los objetos de tipo String son inmutables: cada operación que «modifica» una cadena genera en realidad un nuevo objeto. Para realizar muchas modificaciones se recomienda usar StringBuffer o StringBuilder, que sí permiten cambiar el contenido.

Algunos métodos útiles son:

  • length(): devuelve el número de caracteres.
  • charAt(int index): devuelve el carácter de una posición.
  • toCharArray(): devuelve un array de caracteres con el contenido de la cadena.

Métodos frecuentes de String:

  • Acceso Caracteres individuales: charAt(int index)
  • Comparación Comparar cadenas: equals(), equalsIgnoreCase(), compareTo(), compareToIgnoreCase()
  • Búsqueda Buscar subcadenas o caracteres: indexOf(), lastIndexOf(), contains()
  • Extracción Extraer subcadenas: substring()
  • Reemplazo Reemplazar subcadenas: replace(), replaceAll(), replaceFirst()
  • Formato Mayúsculas / minúsculas: toUpperCase(), toLowerCase()
  • División Dividir una cadena con un separador: split(String regex)
  • Limpieza Quitar espacios al inicio y al final: trim()
Error típico: usar == para comparar cadenas.
Recuerda: == compara referencias, mientras que equals() compara el contenido de las cadenas.
Tip: Si vas a modificar una cadena dentro de un bucle (por ejemplo, concatenando muchas veces), considera usar StringBuilder para mejorar el rendimiento.

Además, la clase String dispone del método estático valueOf(), sobrecargado para aceptar parámetros de todos los tipos Java y devolver una cadena con una representación textual del valor recibido.

Para más información detallada, consulta la documentación oficial: API oficial de String

StringBuilder es una clase de Java que permite trabajar con cadenas modificables (mutables). A diferencia de String, las operaciones como append(), insert() o delete() modifican el mismo objeto sin crear nuevas cadenas en memoria.

Es especialmente útil cuando realizamos muchas concatenaciones dentro de un bucle, ya que mejora el rendimiento frente al uso repetido del operador + con String.

Constructores más habituales:

  • new StringBuilder(): crea un buffer vacío.
  • new StringBuilder(String str): crea un buffer inicializado con str.
  • new StringBuilder(int capacity): crea un buffer con capacidad inicial indicada.

Métodos frecuentes de StringBuilder:

  • append(x): añade al final (cadenas, números, etc.).
  • insert(int offset, x): inserta en una posición.
  • delete(int start, int end): elimina un rango de caracteres.
  • replace(int start, int end, String str): sustituye un tramo.
  • reverse(): invierte el contenido.
  • toString(): devuelve un String con el contenido actual.
Rendimiento: Para muchas concatenaciones (por ejemplo, construir una cadena dentro de un bucle), StringBuilder suele ser más eficiente que usar String con +.

Ejemplos básicos de uso de StringBuilder:

Descripción Código Java Resultado
Construir una cadena con append() 

StringBuilder sb = new StringBuilder();
sb.append("Hola");
sb.append(" ");
sb.append("mundo");
System.out.println(sb.toString());
Hola mundo
Insertar texto en medio de la cadena 

StringBuilder sb = new StringBuilder("Hola mundo");
sb.insert(5, "Java ");
System.out.println(sb.toString());
Hola Java mundo
Eliminar parte de la cadena con delete() 

StringBuilder sb = new StringBuilder("123-456-789");
sb.delete(3, 4); // elimina el guion
System.out.println(sb.toString());
123456-789
Invertir una cadena con reverse() (mini-reto) 

StringBuilder sb = new StringBuilder("programa");
sb.reverse();
System.out.println(sb.toString());
