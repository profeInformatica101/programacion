Documentación oficial

operación intermedia · filter 
List<Integer> numeros = List.of(1, 2, 3, 4, 5, 6, 8, 11, 14);

List<Integer> pares = numeros.stream()
    .filter(n -> n % 2 == 0)
    .toList();

System.out.println("Números pares: " + pares);
▶️ Salida esperada:
Números pares: [2, 4, 6, 8, 14]
filter() evalúa un predicado (lambda) y mantiene los elementos que cumplen la condición.

map · transformación 1:1 
List<String> nombres = List.of("ana", "juan", "pedro", "sofía");

List<String> mayusculas = nombres.stream()
    .map(String::toUpperCase)
    .toList();

// También capitalización elegante
List<String> capitalizadas = nombres.stream()
    .map(n -> n.substring(0,1).toUpperCase() + n.substring(1))
    .toList();

System.out.println("MAYÚSCULAS: " + mayusculas);
System.out.println("Capitalizadas: " + capitalizadas);
▶️ Salida esperada:
MAYÚSCULAS: [ANA, JUAN, PEDRO, SOFÍA]
Capitalizadas: [Ana, Juan, Pedro, Sofía]
map() aplica una función a cada elemento, ideal para extraer campos o modificar.

reduce · operación terminal 
List<Integer> valores = List.of(2, 4, 6, 8);

int suma = valores.stream()
    .reduce(0, Integer::sum);

int producto = valores.stream()
    .reduce(1, (a, b) -> a * b);

// Máximo sin usar max()
int maximo = valores.stream()
    .reduce(Integer::max)
    .orElseThrow();

System.out.println("Suma: " + suma);
System.out.println("Producto: " + producto);
System.out.println("Máximo: " + maximo);
▶️ Salida esperada:
Suma: 20
Producto: 384
Máximo: 8
reduce() combina los elementos en un único resultado acumulativo.

sorted · limit · skip 
List<Integer> numeros = List.of(42, 3, 17, 9, 88, 23, 5);

List<Integer> top3Ascendente = numeros.stream()
    .sorted()
    .limit(3)
    .toList();

// Saltar los 2 más pequeños y tomar siguientes 2
List<Integer> saltarDos = numeros.stream()
    .sorted()
    .skip(2)
    .limit(2)
    .toList();

System.out.println("Top 3 menores: " + top3Ascendente);
System.out.println("Después de saltar 2: " + saltarDos);
▶️ Salida esperada:
Top 3 menores: [3, 5, 9]
Después de saltar 2: [9, 17]
⚡ Composición fluida: sorted() → limit() → toList()

POJO · filtros compuestos 
record Persona(String nombre, int edad, String ciudad) {}

List<Persona> personas = List.of(
    new Persona("Ana", 20, "Madrid"),
    new Persona("Luis", 30, "Barcelona"),
    new Persona("Carla", 25, "Madrid"),
    new Persona("Pedro", 35, "Valencia"),
    new Persona("Elena", 22, "Madrid")
);

// Personas mayores de 22 años que viven en Madrid
List<String> madrileñosMayores = personas.stream()
    .filter(p -> p.edad() > 22)
    .filter(p -> p.ciudad().equals("Madrid"))
    .map(Persona::nombre)
    .toList();

// Estadísticas: edad promedio usando mapToInt
double edadPromedio = personas.stream()
    .mapToInt(Persona::edad)
    .average()
    .orElse(0.0);

System.out.println("Madrileños mayores de 22: " + madrileñosMayores);
System.out.printf("Edad promedio del grupo: %.1f años%n", edadPromedio);
▶️ Salida esperada:
Madrileños mayores de 22: [Carla]
Edad promedio del grupo: 26.4 años
🔥 Tip extra: Usa mapToInt() + average() para estadísticas eficientes.

Collectors · groupingBy 
import java.util.stream.Collectors;

record Producto(String nombre, String categoria, double precio) {}

List<Producto> productos = List.of(
    new Producto("Laptop", "Electrónica", 850.0),
    new Producto("Mouse", "Electrónica", 25.5),
    new Producto("Silla", "Oficina", 120.0),
    new Producto("Mesa", "Oficina", 210.0),
    new Producto("Cuaderno", "Papelería", 5.99)
);

// Agrupar productos por categoría
Map<String, List<Producto>> porCategoria = productos.stream()
    .collect(Collectors.groupingBy(Producto::categoria));

// Contar productos por categoría
Map<String, Long> conteo = productos.stream()
    .collect(Collectors.groupingBy(Producto::categoria, Collectors.counting()));

System.out.println("Productos por categoría: " + porCategoria.keySet());
System.out.println("Conteo: " + conteo);
▶️ Salida esperada:
Productos por categoría: [Electrónica, Oficina, Papelería]
Conteo: {Electrónica=2, Oficina=2, Papelería=1}
📌 groupingBy es ideal para reportes y análisis de datos.

flatMap · colecciones anidadas 
List<List<Integer>> listaDeListas = List.of(
    List.of(1, 2, 3),
    List.of(4, 5),
    List.of(6, 7, 8, 9)
);

List<Integer> listaPlana = listaDeListas.stream()
    .flatMap(List::stream)
    .toList();

// Sumar solo pares después de aplanar
int sumaPares = listaDeListas.stream()
    .flatMap(List::stream)
    .filter(n -> n % 2 == 0)
    .mapToInt(Integer::intValue)
    .sum();

System.out.println("Lista plana: " + listaPlana);
System.out.println("Suma de pares después flatMap: " + sumaPares);
▶️ Salida esperada:
Lista plana: [1, 2, 3, 4, 5, 6, 7, 8, 9]
Suma de pares después flatMap: 20
🚀 flatMap → "aplanamiento" de estructuras multidimensionales.
Cheat Sheet visual
  • filter(Predicate) → selecciona elementos
  • map(Function) → transforma cada elemento
  • flatMap() → aplana + transforma
  • sorted() / limit() / skip() → orden y paginación
  • collect(Collectors.toList()) / toList() → recolectar
Pro tip del día

Los Streams son lazy: las operaciones intermedias no se ejecutan hasta que se invoca una operación terminal (forEach, collect, reduce, etc.). ¡Aprovéchalo para cadenas eficientes!
