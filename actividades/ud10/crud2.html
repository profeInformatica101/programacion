Redacción para el alumnado

Desarrolla una aplicación en Java para gestionar tareas académicas de un estudiante de DAW. La aplicación funcionará por consola y permitirá realizar un CRUD completo. Los datos deberán almacenarse en un fichero de texto llamado tareas.txt. Cada tarea incluirá un identificador, un título, un módulo y un estado. El programa deberá usar interfaces, clases, flujos de caracteres y salida formateada en consola.

Crud.java 
// Contrato común para operaciones CRUD
public interface Crud<T> {
    void    crear(T objeto);
    T       buscarPorId(int id);
    T[]     listar();
    boolean actualizar(int id, T nuevo);
    boolean eliminar(int id);
}
Tarea.java 
public class Tarea {
    private int id;
    private String titulo;
    private String modulo;
    private String estado;

    public Tarea(int id, String titulo, String modulo, String estado) {
        this.id = id;
        this.titulo = titulo;
        this.modulo = modulo;
        this.estado = estado;
    }

    public int getId() { return id; }
    public void setId(int id) { this.id = id; }

    public String getTitulo() { return titulo; }
    public void setTitulo(String titulo) { this.titulo = titulo; }

    public String getModulo() { return modulo; }
    public void setModulo(String modulo) { this.modulo = modulo; }

    public String getEstado() { return estado; }
    public void setEstado(String estado) { this.estado = estado; }

    @Override
    public String toString() {
        return String.format(
            "%-4d | %-25s | %-20s | %-12s",
            id, titulo, modulo, estado);
    }

    public String toLinea() {
        return id + ";" + titulo + ";" + modulo + ";" + estado;
    }

    public static Tarea desdeLinea(String linea) {
        String[] p = linea.split(";");
        return new Tarea(
            Integer.parseInt(p[0]),
            p[1], p[2], p[3]
        );
    }
}

Contenidos del tema que se trabajan

Flujos de caracteres aplicados a una situación real
Apertura y cierre

Uso de try-with-resources. Java cierra el flujo automáticamente al salir del bloque.

Modos de acceso

new FileWriter("tareas.txt", true) abre en modo append: añade sin borrar el contenido previo.

Lectura eficiente

BufferedReader envuelve a FileReader para leer líneas completas con readLine().

Escritura estructurada

Cada tarea ocupa una línea separada por ;. Es sencillo de guardar, revisar y parsear.
