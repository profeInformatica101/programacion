Boletín 4 · Funciones (Java)

Unidad 4 — Ejercicios con funciones, parámetros, valor devuelto, sobrecarga y recursividad.

Diseñar la función eco() a la que se le pasa como parámetro un número n y muestra por pantalla n veces el mensaje “Eco ...”.

Escribir una función a la que se le pasen dos enteros y muestre todos los números comprendidos entre ellos.

Realizar una función que calcule y muestre el área o el volumen de un cilindro, según se especifique.
Opción 1: área. Opción 2: volumen. Además se pasa el radio de la base y la altura.
Fórmulas:
área = 2 · π · r · (h + r)
volumen = π · r² · h

Diseñar una función que reciba como parámetros dos números enteros y que devuelva el mayor de los dos.

Repite el ejercicio anterior con una versión que calcule el máximo de 3 números.

Crear una función que, mediante un booleano, indique si el carácter pasado como parámetro corresponde a una vocal.

Diseñar una función que determine si un número es primo.

Escribir una función que, dado un entero, devuelva el número de divisores primos que tiene.

Implementar la función divisoresPrimos() que muestre por consola todos los divisores primos del número pasado como parámetro.

Escribir una función que decida si dos enteros positivos son amigos. Dos números son amigos si la suma de sus divisores propios (distintos de ellos mismos) es igual.

Diseñar una función que calcule a^n, donde a es real y n entero no negativo. Realiza versión iterativa y versión recursiva.

Diseñar la función calculadora() a la que se le pasan dos enteros (operandos) y la operación a realizar: 1 suma, 2 resta, 3 multiplicación, 4 división. La función devolverá un número real como resultado.

Calcular el factorial de n recursivamente. Recuerda que por definición 0! = 1.

Diseñar una función que calcule el n-ésimo término de la serie de Fibonacci:
f(0) = 1, f(1) = 1 y f(n) = f(n-1) + f(n-2) para n ≥ 2.
