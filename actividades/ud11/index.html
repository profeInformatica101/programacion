Ejercicios de Ficheros Binarios en Java

Leer un número decimal desde consola y guardarlo usando DataOutputStream.

Utiliza FileOutputStream y luego DataOutputStream.writeDouble(valor). Cierra ambos streams al terminar.

Leer el número previamente guardado y mostrarlo en consola usando DataInputStream.

Usa FileInputStream y DataInputStream.readDouble(). Maneja EOFException si es necesario.

Solicitar números enteros positivos y guardarlos en un archivo binario. Terminar al introducir un número negativo y luego mostrar todos los números.

Usa DataOutputStream.writeInt() para guardar y DataInputStream.readInt() para leer. Bucle while(num >= 0).

Pedir un entero n, luego n números double, guardarlos en un array y almacenar la tabla en un archivo binario.

Usa un for para leer los doubles y DataOutputStream.writeDouble(). Para leer, recorre con for y readDouble().

Leer un array de doubles desde un archivo binario y mostrarlo por consola.

Guarda primero el tamaño con writeInt() para poder leer correctamente con readInt().

Usar un bucle hasta EOFException para leer todos los números double de un archivo binario desconocido.

Coloca la lectura dentro de un try con catch (EOFException e) para detectar el final del archivo.

Escribir una frase, guardarla en un archivo binario y recuperarla.

Usa writeUTF() para guardar y readUTF() para leer. Maneja excepciones de IO.

Guardar líneas de texto en un archivo binario hasta línea vacía. Leer y mostrar todo el texto.

Usa writeUTF(linea) para cada línea. Para leer, bucle while(true) con readUTF() y EOFException.

Generar 10 enteros aleatorios <100, ordenarlos y guardarlos en un archivo binario. Recuperarlos y mostrarlos.

Usa Random.nextInt(100) para generar los números. Para ordenar, Arrays.sort(array). Luego guarda con writeInt().

Guardar número de llamadas por jornada en un archivo binario. Menú para añadir o ver registros.

Crear clase Registro con Serializable. Usa ObjectOutputStream y ObjectInputStream para persistencia.

Crear clase Persona con nombre, edad y email. Guardar varios objetos en archivo binario.

Implementa Serializable en Persona. Usa ObjectOutputStream.writeObject(objeto) y ObjectInputStream.readObject().

Leer todos los objetos Persona guardados previamente y mostrarlos por consola.

Usa bucle while(true) con readObject() y captura EOFException para terminar la lectura.

Crear clase Producto con código, nombre y precio. Guardar varios productos usando serialización.

Clase Producto implementa Serializable. Usa ObjectOutputStream.writeObject() y ObjectInputStream.readObject().

Añadir nuevos productos a un archivo binario que ya contiene productos.

Para añadir objetos a archivo existente, usa FileOutputStream(file,true) con cuidado: si usas ObjectOutputStream directamente, puede corromper el stream. Investiga “Append ObjectOutputStream” para Java.

Leer lista de productos y permitir modificar el precio o nombre de un producto identificado por código.

Lee todos los objetos a una lista temporal, modifica el que corresponda, y luego vuelve a escribir todo el archivo con ObjectOutputStream.

Implementar un CRUD completo para la clase Producto: Crear, Leer, Actualizar y Borrar objetos en un archivo binario.

Usa una lista temporal para almacenar todos los objetos mientras realizas operaciones CRUD. Cada acción (alta, baja, modificación) requiere reescribir el archivo binario completo con ObjectOutputStream. Implementa un menú interactivo para cada opción.
