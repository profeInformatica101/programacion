Primer Stream forEach

Recorrido declarativo

▾
🎯 Objetivo: Introducción al pipeline de streams.

stream() crea una secuencia. forEach es terminal e imprime cada elemento.

List<Integer> nums = List.of(3, 7, 1, 9);
// Muestra cada número
nums.stream().forEach(System.out::println);

🚀 Proyecto final · Analizador de texto

Integración total: lectura, limpieza, frecuencias, top 10

▾
🎯 OBJETIVO MASTER: Construir un contador de palabras profesional con streams puros.
  • Leer fichero con Files.lines
  • Normalizar a minúsculas y dividir con expresiones regulares
  • Filtrar stopwords (el, la, de, y, a, con)
  • Agrupar por palabra y contar frecuencias
  • Ordenar descendente y mostrar TOP 10
Set<String> stopwords = Set.of("el","la","de","y","a","en");
try (Stream<String> lines = Files.lines(Path.of("texto.txt"))) {
    lines.map(String::toLowerCase)
         .flatMap(l -> Arrays.stream(l.split("[^a-záéíóúüñ]+")))
         .filter(w -> w.length() > 1 && !stopwords.contains(w))
         .collect(Collectors.groupingBy(Function.identity(), Collectors.counting()))
         .entrySet().stream()
         .sorted(Map.Entry.<String,Long>comparingByValue().reversed())
         .limit(10)
         .forEach(e -> System.out.printf("%-15s → %d%n", e.getKey(), e.getValue()));
}
